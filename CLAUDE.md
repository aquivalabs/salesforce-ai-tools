# salesforce-ai-tools

Reusable GitHub Actions workflows and Claude Code skills for AI-assisted Salesforce development. See [README.md](README.md) for setup and usage instructions.

## Skills

- `sf-ticket-to-pr` — turns GitHub issues into pull requests via an @butler mention
- `agentforce` / `agentforce-deploy` — test and deploy Agentforce metadata
- `playwright-sf` — headless Salesforce UI automation for change verification
- `sf-code-analyzer` — runs Salesforce Code Analyzer on changed files
